## 📌 啟用 GitHub Pages（手動設定）

### 步驟：

1. 訪問倉庫設定頁面：
   👉 https://github.com/ken0900101318-beep/one-quote-system/settings/pages

2. 在「Build and deployment」區塊：
   - **Source**: Deploy from a branch
   - **Branch**: main
   - **Folder**: / (root)

3. 點擊「Save」

4. 等待 1-2 分鐘後，系統會顯示網址：
   👉 **https://ken0900101318-beep.github.io/one-quote-system/**
