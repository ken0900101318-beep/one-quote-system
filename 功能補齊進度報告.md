## 🧪 測試步驟

### 測試已完成功能（1-3）

1. **清除快取**
   ```
   F12 → Console
   localStorage.clear()
   重新載入
   ```

2. **登入系統**
   ```
   阿建 / admin123
   ```

3. **測試刪除專案**
   - 儀表板 → 找到任一專案
   - 點擊「🗑️ 刪除」
   - 確認對話框
   - 專案消失

4. **測試報價項目刪除**
   - 進入任一專案
   - 點擊「📝 修改報價數量」
   - 點擊任一項目的「🗑️」
   - 項目變透明
   - 總額自動扣減
   - 儲存變更

5. **測試報價項目新增**
   - 修改報價 Modal
   - 選擇項目（下拉選單）
   - 輸入數量
   - 單價自動帶入
   - 點擊「➕ 加入」
   - 項目出現在清單（綠色）
   - 總額自動增加
   - 儲存變更
