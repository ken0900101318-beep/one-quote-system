# 報價系統功能補齊計畫

## 立即實作（3 個高優先級功能）

### 1. 刪除專案 ✅
**位置**：dashboard.html  
**功能**：
- 專案列表的操作欄新增「🗑️ 刪除」按鈕
- 確認對話框（防止誤刪）
- 主管權限才能刪除
- 刪除後無法復原（或移到回收站）

**程式碼位置**：
```javascript
// dashboard.html 中加入
function deleteProject(projectId) {
    if (!confirm('確定要刪除此專案嗎？刪除後無法復原！')) return;
    
    const projects = JSON.parse(localStorage.getItem('projects') || '[]');
    const filtered = projects.filter(p => p.id !== projectId);
    localStorage.setItem('projects', JSON.stringify(filtered));
    
    alert('專案已刪除！');
    location.reload();
}
```

### 2. 報價中刪除項目 ✅
**位置**：project.html - quoteEditModal  
**功能**：
- 報價項目清單每一行加入「🗑️」按鈕
- 點擊刪除該項目
- 自動重新計算總額
- 建立新版本

**表格欄位更新**：
```
項目名稱 | 原數量 | 新數量 | 原單價 | 新單價 | 新小計 | [🗑️]
```

---

### 3. 報價中新增項目 ✅
**位置**：project.html - quoteEditModal  
**功能**：
- Modal 底部新增「➕ 新增項目」按鈕
- 打開下拉選單（從價格表選擇）
- 輸入數量
- 自動計算小計
- 加入清單並重新計算總額

**新增流程**：
1. 點擊「➕ 新增項目」
2. 彈出小型表單：
   - 選擇項目（下拉選單）
   - 輸入數量
   - 單價（自動帶入，可修改）
3. 點擊「加入」→ 加到清單
4. 儲存時建立新版本
