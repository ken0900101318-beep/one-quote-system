### **功能 5 + 7：刪除和回復報價版本**
**需要加入的位置：** 專案詳情頁面 - 報價版本顯示區

**建議實作方式：**
在報價明細下方加入「報價版本歷史」區塊：

```html
<h3 style="margin-top: 40px;">📋 報價版本歷史</h3>
<div id="quoteVersionsHistory"></div>
```

在 JavaScript 中渲染版本列表：
```javascript
const quoteVersionsHtml = project.quoteVersions.map((ver, index) => {
    const isCurrent = index === project.quoteVersions.length - 1;
    const canDelete = index > 0 && !isCurrent;
    
    return `
        <div style="padding: 15px; border: 1px solid #e5e7eb; border-radius: 8px; margin-bottom: 10px; ${isCurrent ? 'background: #f0fdf4; border-color: #059669;' : ''}">
            <div style="display: flex; justify-content: space-between; align-items: center;">
                <div>
                    <strong>${ver.version}</strong> ${isCurrent ? '<span style="color: #059669;">（當前版本）</span>' : ''}
                    <br>日期：${ver.date} | 總額：$${ver.totalAmount.toLocaleString()}
                    <br>原因：${ver.reason}
                </div>
                <div>
                    ${!isCurrent ? `
                        <button onclick="restoreQuoteVersion('${projectId}', ${index})" 
                                class="btn btn-primary" 
                                style="padding: 8px 16px; margin-right: 5px;">
                            🔄 回復到此版本
                        </button>
                    ` : ''}
                    ${canDelete ? `
                        <button onclick="deleteQuoteVersion('${projectId}', ${index})" 
                                class="btn" 
                                style="padding: 8px 16px; background: #ef4444; color: white;">
                            🗑️ 刪除
                        </button>
                    ` : ''}
                </div>
            </div>
        </div>
    `;
}).join('');

document.getElementById('quoteVersionsHistory').innerHTML = quoteVersionsHtml;
```
