### 📋 doGet() 支援的 API

| API Action | 參數 | 回傳格式 | 說明 |
|-----------|------|---------|------|
| `test` | - | `{ success, message, timestamp }` | API 健康檢查 |
| `getUsers` | - | `{ success, users[] }` | 取得所有員工（從員工管理系統） |
| `getProjects` | - | `{ success, projects[] }` | 取得所有專案 |
| `getProject` | `id` | `{ success, project }` | 取得單一專案 |
| `getPriceTable` | - | `{ success, items[] }` | 取得價目表 |
| `getCategories` | - | `{ success, categories[] }` | 取得分類 |
| `addProject` | `data` (JSON string) | `{ success, message, id }` | 新增專案（GET 方式，不建議） |
| `updateProject` | `id`, `data` (JSON string) | `{ success, message }` | 更新專案（GET 方式，不建議） |
| `deleteProject` | `id` | `{ success, message }` | 刪除專案（GET 方式，不建議） |

### 📋 doPost() 支援的 API

| API Action | 參數（POST Body） | 回傳格式 | 說明 |
|-----------|------------------|---------|------|
| `importPriceTable` | `{ action, items[] }` | `{ success, message, count }` | 批次匯入價目表 |
| `addProject` | `{ action, project }` | `{ success, message, id }` | 新增專案（POST 方式，建議） |

### 🔑 重要發現

1. **GET vs POST 混用問題**：`addProject`、`updateProject`、`deleteProject` 同時支援 GET 和 POST，但：
   - GET 方式使用 URL 參數傳遞 JSON，有長度限制（約 2KB）
   - POST 方式更安全、無長度限制
   - **建議統一使用 POST**

2. **參數格式不一致**：
   - GET 方式：`data` 需要 `JSON.stringify()`
   - POST 方式：直接傳遞物件

3. **缺少的 API**：
   - ❌ `updatePriceTable`（更新單一價目）
   - ❌ `deletePriceTable`（刪除價目）
   - ❌ `addCategory`（新增分類）
   - ❌ `updateCategory`（更新分類）
   - ❌ `deleteCategory`（刪除分類）

## 📋 發現的問題清單

### 🔴 高優先級問題（影響功能）

1. **API 方式錯誤 - addProject 使用 GET**
   - **位置**：`api-quote.js` 的 `addProject()` 方法
   - **問題**：使用 GET + URL 參數傳遞 JSON，當報價項目多時會超過 URL 長度限制（約 2KB）
   - **影響**：新增包含多個項目的報價時會失敗
   - **修復方式**：改用 POST 方式，傳送完整 JSON Body

2. **缺少編輯報價功能**
   - **位置**：前端缺少 `edit-quote.html`
   - **問題**：無法修改已建立的報價單
   - **影響**：使用者只能刪除重建，無法微調報價
   - **修復方式**：
     - 建立 `edit-quote.html`（可複用 `create-quote-simple.html` 的程式碼）
     - 在 `api-quote.js` 加入 POST 版本的 `updateProject()`
     - 在 dashboard 和 view-quote 加入「編輯」按鈕

3. **缺少價目表管理功能**
   - **位置**：前端完全缺少
   - **問題**：無法在系統內管理價目表，只能直接編輯 Google Sheets
   - **影響**：使用體驗不佳，需要技術知識
   - **修復方式**：
     - 建立 `price-table.html`（價目表管理頁面）
     - 在後端加入 `addPriceItem`、`updatePriceItem`、`deletePriceItem` API
     - 支援批次匯入（已有 `importPriceTable` 但前端未使用）

### 🟡 中優先級問題（影響使用體驗）

4. **updateProject 使用 GET**
   - **位置**：`api-quote.js` 的 `updateProject()` 方法
   - **問題**：同上，URL 長度限制
   - **影響**：未來實作編輯功能時會遇到問題
   - **修復方式**：改用 POST 方式

5. **缺少快速選擇價目功能**
   - **位置**：`create-quote-simple.html`
   - **問題**：新增項目時無法從價目表快速選擇
   - **影響**：需要手動輸入所有欄位，效率低
   - **修復方式**：
     - 在新增項目時呼叫 `getPriceTable()`
     - 加入「從價目表選擇」按鈕或下拉選單
     - 選擇後自動填入分類、單價、單位

6. **缺少「單位」欄位**
   - **位置**：`create-quote-simple.html`
   - **問題**：前端沒有「單位」輸入欄位（例如：個、組、式、坪）
   - **影響**：報價單缺少單位資訊，不專業
   - **修復方式**：在項目表單加入「單位」輸入欄位

7. **缺少分類管理功能**
   - **位置**：前端完全缺少
   - **問題**：無法在系統內管理分類
   - **影響**：新增分類需要直接編輯 Google Sheets

     - 建立 `categories.html`（分類管理頁面）

```javascript
const QuoteAPI = {
    // ... 其他方法保持不變 ...
    
    /**
     * 發送 POST 請求（新增）
     */
    async requestPost(action, data, options = {}) {
        const {
            timeout = 30000,
            retries = 1
        } = options;
        
        let lastError;
        
        for (let attempt = 0; attempt <= retries; attempt++) {
            try {
                console.log(`API POST 請求 (嘗試 ${attempt + 1}/${retries + 1}):`, action, data);
                
                const controller = new AbortController();
                const timeoutId = setTimeout(() => controller.abort(), timeout);
                
                const response = await fetch(QUOTE_API.endpoint, {
                    method: 'POST',
                    headers: {
                        'Content-Type': 'application/json'
                    },
                    body: JSON.stringify({
                        action: action,
                        ...data
                    }),
                    signal: controller.signal
                });
                
                clearTimeout(timeoutId);
                
                if (!response.ok) {
                    throw new Error(`HTTP ${response.status}: ${response.statusText}`);
                }
                
                let result;
                try {
                    result = await response.json();
                } catch (parseError) {
                    throw new Error('伺服器回應格式錯誤');
                }
                
                console.log('API 回應:', result);
                
                if (result.success === false && result.error) {
                    if (result.error.includes('找不到') || 
                        result.error.includes('驗證失敗') ||
                        result.error.includes('權限不足')) {
                        return result;
                    }
                    throw new Error(result.error);
                }
                
                return result;
                
            } catch (error) {
                lastError = error;
                
                if (error.name === 'AbortError') {
                    lastError = new Error('連線超時，請檢查網路狀態');
                }
                
                if (error.message.includes('Failed to fetch')) {
                    lastError = new Error('無法連線到伺服器，請檢查網路狀態');
                }
                
                if (attempt < retries) {
                    console.log(`等待 ${(attempt + 1) * 1000}ms 後重試...`);
                    await new Promise(resolve => setTimeout(resolve, (attempt + 1) * 1000));
                    continue;
                }
                
                break;
            }
        }
        
        return {
            success: false,
            error: lastError.message || '未知錯誤',
            details: lastError
        };
    },
    
    // 新增專案（改用 POST）
    async addProject(data) {
        if (!data) {
            return { success: false, error: '缺少專案資料' };
        }
        
        if (!data.customerName || !data.phone) {
```

```javascript
/**
 * 處理 POST 請求（改進版）
 */
function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const action = data.action || 'importPriceTable';
    
    let result;
    
    switch (action) {
      case 'importPriceTable':
        result = importPriceTable(data.items || []);
        break;
        
      case 'addProject':
        result = addProject(data.project || {});
        break;
        
      case 'updateProject':
        result = updateProject(data.id, data.project || {});
        break;
        
      default:
        result = { success: false, error: '不支援的操作' };
    }
    
    return ContentService
      .createTextOutput(JSON.stringify(result))
      .setMimeType(ContentService.MimeType.JSON);
  } catch (error) {
    return ContentService
      .createTextOutput(JSON.stringify({ 
        success: false, 
        error: error.toString() 
      }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

**修改檔案**：`create-quote-simple.html`

```javascript
let priceTable = []; // 儲存價目表資料

// 載入價目表
async function loadPriceTable() {
    try {
        const result = await QuoteAPI.getPriceTable();
        if (result.success && result.items) {
            priceTable = result.items;
            console.log('已載入價目表:', priceTable.length, '項');
        }
    } catch (error) {
        console.error('載入價目表失敗:', error);
    }
}

// 在新增項目時加入「從價目表選擇」按鈕
<div class="form-row">
    <label>項目名稱 *</label>
    <div style="display: flex; gap: 10px;">
        <input type="text" id="itemName${itemCount}" required maxlength="100" style="flex: 1;">
        <button type="button" class="btn btn-sm btn-secondary" onclick="selectFromPriceTable(${itemCount})">
            📋 從價目表選擇
        </button>
    </div>
</div>

// 選擇價目函式
function selectFromPriceTable(itemId) {
    // 建立簡單的彈出選單
    const itemName = prompt('請輸入項目關鍵字搜尋：');
    if (!itemName) return;
    
    const filtered = priceTable.filter(item => 
        item.name.includes(itemName)
    );
    
    if (filtered.length === 0) {
        alert('找不到相關項目');
        return;
    }
    
    // 如果只有一個結果，直接填入
    if (filtered.length === 1) {
        fillPriceItem(itemId, filtered[0]);
        return;
    }
    
    // 多個結果，顯示列表供選擇
    const options = filtered.map((item, idx) => 
        `${idx + 1}. ${item.name} - ${item.price} 元/${item.unit}`
    ).join('\n');
    
    const choice = prompt(`找到 ${filtered.length} 項結果：\n${options}\n\n請輸入編號：`);
    const index = parseInt(choice) - 1;
    
    if (index >= 0 && index < filtered.length) {
        fillPriceItem(itemId, filtered[index]);
    }
}

// 填入價目資料
function fillPriceItem(itemId, priceItem) {
    document.getElementById(`itemName${itemId}`).value = priceItem.name;
    document.getElementById(`itemCategory${itemId}`).value = priceItem.category || '';
    document.getElementById(`itemSubCategory${itemId}`).value = priceItem.subCategory || '';
    document.getElementById(`itemPrice${itemId}`).value = priceItem.price || 0;
    document.getElementById(`itemUnit${itemId}`).value = priceItem.unit || '';
    calculateTotal();
    Toast.success('已填入價目資料');
}

// 頁面載入時呼叫
(async function init() {
    await loadCategories();
    await loadPriceTable(); // 加入這行
    addItem();
})();
```
