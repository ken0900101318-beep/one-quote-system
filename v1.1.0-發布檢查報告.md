## 📝 測試場景

### 場景 1：新增專案收款
```
1. 開啟專案詳情頁
2. 點擊「新增收款」
3. 填寫：日期（今天）、類型（訂金）、金額（100,000）、方式（匯款）
4. 點擊儲存
5. 檢查：收款記錄表格出現、已收金額 $100,000、進度條 12%（假設總價 $856,800）
```

### 場景 2：修改專案資訊
```
1. 點擊「編輯資訊」
2. 修改店名為「高雄左營店（測試）」
3. 修改狀態為「已簽約」
4. 點擊儲存
5. 檢查：標題更新、狀態徽章變更
```

### 場景 3：新增報價項目
```
1. 點擊「新增項目」
2. 選擇「壁紙 - $850」
3. 數量輸入 10
4. 檢查小計自動計算 $8,500
5. 點擊新增
6. 檢查：項目加入表格、總額增加 $8,500
```
