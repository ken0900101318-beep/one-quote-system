## 🔢 計算邏輯

### 1️⃣ 總成本計算
```
遍歷報價項目：
  - 從價格表（priceTable）匹配項目名稱
  - 找到對應項目的成本（cost）
  - 項目成本 = 數量 × 單位成本
  - 總成本 = Σ(所有項目成本)
  - 若成本為 0 則跳過不計

範例：
輕隔間(60K) × 10坪 × $3,000/坪 = $30,000
輕鋼架 × 54坪 × $1,000/坪 = $54,000
加盟費用 × 1式 × $0/式 = $0 (跳過)
...
總成本 = $30,000 + $54,000 + ... = $1,156,000
```

### 2️⃣ 毛利計算
```
毛利 = 報價總額 - 總成本
     = $1,924,100 - $1,156,000
     = $768,100
```

### 3️⃣ 毛利率計算
```
毛利率 = (毛利 / 報價總額) × 100%
       = ($768,100 / $1,924,100) × 100%
       = 39.9%
```

### 4️⃣ 成本占比計算
```
成本占比 = (總成本 / 報價總額) × 100%
         = ($1,156,000 / $1,924,100) × 100%
         = 60.1%
```

## ⚠️ 重要提醒

### 1️⃣ 成本資料必須正確
**問題**：價格表中的成本欄位若為 0 或不正確，會導致獲利分析失準

**解決**：
1. 進入「價格管理」頁面
2. 檢查每個項目的「成本」欄位
3. 確保成本資料正確填寫

**檢查清單**：
```
✅ 輕隔間(60K)：成本 $3,000/坪（已填寫）
✅ 輕鋼架：成本 $1,000/坪（已填寫）
✅ 入門款地板：成本 $1,000/坪（已填寫）
✅ 一般壁紙：成本 $600/坪（已填寫）
❓ 加盟費用：成本 $0/式（系統費用，無成本）
❓ 智能門鎖：成本 $0/組（設備費用，成本待確認）
❓ 冷氣：成本 $0/台（設備費用，成本待確認）
```

### 2️⃣ 系統費用的成本
**特殊情況**：
- 加盟費用：通常成本為 0（純利潤）
- 智能門鎖、麻將桌等設備：可能有採購成本
- 如果有成本，請在價格表中填寫

### 3️⃣ 項目名稱必須匹配
**原理**：系統通過項目名稱從價格表查找成本

**注意**：
- ✅ 報價項目名稱 = 價格表名稱 → 成本正確
- ❌ 報價項目名稱 ≠ 價格表名稱 → 成本為 0（找不到）

**範例**：
```
✅ 報價：輕隔間(60K)
   價格表：輕隔間(60K)
   → 找到成本 $3,000

❌ 報價：輕隔間 60K
   價格表：輕隔間(60K)
   → 找不到，成本 $0
```
