### 立即可測試（3 個功能）

#### 1️⃣ 測試刪除專案
```
步驟：
1. 儀表板 → 找到任一專案
2. 點擊「🗑️ 刪除」按鈕（紅色）
3. 確認對話框會列出將刪除的內容
4. 測試確認/取消功能
```

#### 2️⃣ 測試報價項目刪除
```
步驟：
1. 進入任一專案
2. 點擊「📝 修改報價數量」
3. 找到任一報價項目
4. 點擊該項目的「🗑️」按鈕
5. 項目應該變透明 + 刪除線
6. 總額自動扣減
7. 點擊「確認修改」儲存
```

#### 3️⃣ 測試報價項目新增
```
步驟：
1. 修改報價 Modal
2. 向下滾動到底部
3. 查找「➕ 新增項目」區塊（綠色虛線邊框）
4. 選擇項目（下拉選單）
5. 輸入數量
6. 單價自動帶入
7. 點擊「➕ 加入」
8. 項目應該加入清單（綠色背景）
9. 點擊「確認修改」儲存
```
