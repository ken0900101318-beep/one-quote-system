# ONE桌遊報價管理系統 v1.0

## 📋 系統簡介

完整的專案生命週期管理系統，從報價到收款的全流程管理。

## ✨ 核心功能

### 1. 📊 儀表板
- 專案統計（總數、施工中、收款統計）
- 專案列表（搜尋、篩選）
- 進度追蹤

### 2. 📝 專案管理
- 基本資訊管理
- 報價版本控制（價格快照，歷史報價不受影響）
- 收款記錄追蹤
- 施工進度更新
- 狀態變更日誌

### 3. 💰 價格管理
- 35+ 項目價格資料庫
- 分類瀏覽（工班/ONE桌遊/新系統）
- 搜尋功能

### 4. 👥 權限控制
- **主管**（阿建、建佑）：完整權限（可修改）
- **員工**（小圓、尼克）：基本操作
- **會計**：僅查看權限

## 💡 核心特色

### 🔒 價格快照機制
**解決 Excel 致命缺陷：**
- ❌ Excel：價格更新後，舊報價單自動變動
- ✅ 網頁版：報價建立時「凍結」當下價格，永不改變

### 📅 完整生命週期
```
報價 → 簽約 → 施工 → 完工 → 收款
  ↓      ↓      ↓      ↓      ↓
草稿   已簽約  施工中  已完工  已結清
```

### 💰 智慧收款追蹤
- 自動計算收款進度
- 可視化進度條
- 待收款提醒
- 發票記錄

### 📊 權限分級
- 主管：可修改所有資料
- 員工：基本操作
- 會計：僅查看（保護敏感資訊）

## 📱 使用說明

1. **登入系統**
   - 選擇使用者 + 輸入密碼
   
2. **查看專案**
   - 儀表板 → 點擊專案 → 查看詳情
   
3. **查看報價**
   - 專案詳情 → 報價明細
   - 顯示完整項目、價格、備註
   
4. **查看收款**
   - 專案詳情 → 收款明細
   - 已收/待收金額、收款記錄
   
5. **價格查詢**
   - 價格管理 → 搜尋/篩選項目
