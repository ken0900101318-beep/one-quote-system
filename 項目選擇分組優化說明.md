# 報價選擇項目加入子分類分組

**更新時間**：2026-03-13 20:20  
**功能**：項目選擇加入分類分組  
**狀態**：✅ 已完成並推送

---

## 🔍 問題背景

**用戶反饋**：
> 報價，選擇項目時，要有子分類，不然太多

**問題分析**：
- 價格表有 52 個項目
- 全部列在一個下拉選單裡，太長不好找
- 沒有分類結構，需要一個一個找
- 查找效率低，容易選錯

---

## 🎨 技術細節

### HTML 結構（optgroup）

```html
<select id="newItemSelect">
  <option value="">-- 請選擇項目 --</option>
  
  <optgroup label="🏢 公司">
    <optgroup label="　└ 系統設備">
      <option value="0">　　加盟費用 ($283,200 / 式)</option>
      <option value="1">　　智能門鎖 ($7,000 / 組)</option>
      ...
    </optgroup>
    <optgroup label="　└ 麻將桌">
      <option value="10">　　ONE頂規麻將桌 ($23,800 / 台)</option>
      ...
    </optgroup>
  </optgroup>
  
  <optgroup label="🔨 工班">
    <optgroup label="　└ 輕隔間">
      <option value="22">　　輕隔間(60K) ($3,700 / 坪)</option>
      ...
    </optgroup>
  </optgroup>
</select>
```

### JavaScript 分組邏輯

```javascript
// 按照大分類和子分類分組
const grouped = {};
priceTable.forEach((item, idx) => {
    const category = item.category || '未分類';
    const subCategory = item.subCategory || '其他';
    
    if (!grouped[category]) grouped[category] = {};
    if (!grouped[category][subCategory]) grouped[category][subCategory] = [];
    
    grouped[category][subCategory].push({ ...item, idx });
});

// 先顯示公司，再顯示工班
// 使用 optgroup 嵌套實現分層結構
```
