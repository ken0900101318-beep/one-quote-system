## 🎯 功能總覽

**兩個優惠層級：**
1. **單項優惠**：每個項目可單獨設定業務優惠和公司優惠
2. **整單優惠**：針對所有項目小計設定整單業務優惠和整單公司優惠

**兩種優惠方式：**
1. **固定金額**：直接扣除金額（例如：-$5,000）
2. **折扣比例**：按比例計算（例如：9折 = 扣除 10%）

---

## 📋 單項優惠（項目後方）

### 位置

在步驟 2「選擇項目」的已選項目列表中，每個項目後方有三個優惠按鈕：

```
智能門鎖
單價：$7,000 / 組

[10] 組  $70,000  [💼 業務優惠] [🏢 公司優惠] [🎁 贈送] [移除]
```

### 使用方式

#### 設定業務優惠

**步驟：**
```
1. 點擊「💼 業務優惠」按鈕
2. 選擇優惠方式：
   - 1 = 固定金額
   - 2 = 折扣比例
3. 輸入優惠值
4. 確認
```

**範例 1：固定金額**
```
智能門鎖：10 組 × $7,000 = $70,000

點擊「業務優惠」
↓
選擇：1（固定金額）
↓
輸入：5000
↓
結果：
  原價：$70,000
  業務優惠：-$5,000
  優惠價：$65,000
```

**範例 2：折扣比例**
```
智能門鎖：10 組 × $7,000 = $70,000

點擊「業務優惠」
↓
選擇：2（折扣比例）
↓
輸入：9（9折）
↓
結果：
  原價：$70,000
  業務優惠：-$7,000（10% off）
  優惠價：$63,000
```

#### 設定公司優惠

**步驟：**
```
1. 點擊「🏢 公司優惠」按鈕
2. 選擇優惠方式（同業務優惠）
3. 輸入優惠值
4. 確認
```

**重要！計算基準：**
公司優惠是基於「扣除業務優惠後的金額」計算

**範例：**
```
智能門鎖：10 組 × $7,000 = $70,000
已設定業務優惠：-$5,000
扣除業務優惠後：$65,000

點擊「公司優惠」
↓
選擇：1（固定金額）
↓
輸入：2000
↓
結果：
  原價：$70,000
  業務優惠：-$5,000
  公司優惠：-$2,000
  優惠價：$63,000
```

### UI 顯示

**無優惠：**
```
智能門鎖
單價：$7,000 / 組

[10] 組  $70,000  [💼 業務優惠] [🏢 公司優惠] [🎁 贈送] [移除]
```

**有優惠：**
```
智能門鎖
單價：$7,000 / 組
原價：$70,000
業務優惠：-$5,000
公司優惠：-$2,000

[10] 組  $63,000  [💼 業務優惠] [🏢 公司優惠] [🎁 贈送] [移除]
```

## 📋 整單優惠（總報價底部）

### 位置

在步驟 3「確認報價」的費用總計區域

### 使用方式

#### 設定整單業務優惠

**步驟：**
```
1. 點擊「💼 整單業務優惠」按鈕
2. 選擇優惠方式（固定金額 or 折扣比例）
3. 輸入優惠值
4. 確認
```

**範例：**
```
小計：$500,000（所有項目扣除單項優惠後的總和）

點擊「整單業務優惠」
↓
選擇：1（固定金額）
↓
輸入：10000
↓
結果：
  小計：$500,000
  業務優惠：-$10,000
  總計：$490,000（暫不計公司優惠）
```

#### 設定整單公司優惠

**步驟：**
```
1. 點擊「🏢 整單公司優惠」按鈕
2. 選擇優惠方式
3. 輸入優惠值
4. 確認
```

**重要！計算基準：**
整單公司優惠是基於「小計 - 整單業務優惠」計算

**範例：**
```
小計：$500,000
整單業務優惠：-$10,000
扣除整單業務優惠後：$490,000

點擊「整單公司優惠」
↓
選擇：1（固定金額）
↓
輸入：5000
↓
結果：
  小計：$500,000
  業務優惠：-$10,000
  公司優惠：-$5,000
  總計：$485,000
```

### UI 顯示

**無優惠：**
```
💰 費用總計

項目數量：10 項
小計：$500,000

[💼 整單業務優惠] [🏢 整單公司優惠]

總計：$500,000
```

**有優惠：**
```
💰 費用總計

項目數量：10 項
小計：$500,000
業務優惠：-$10,000 [修改]
公司優惠：-$5,000 [修改]

[💼 整單業務優惠] [🏢 整單公司優惠]

總計：$485,000
```

## 🧮 優惠計算順序（重要！）

```
第 1 層：原價
各項目：數量 × 單價
↓

第 2 層：單項業務優惠
每個項目可設定業務優惠
↓

第 3 層：單項公司優惠
基於（原價 - 業務優惠）計算
↓

第 4 層：小計
所有項目扣除單項優惠後的總和
↓

第 5 層：整單業務優惠
針對小計設定
↓

第 6 層：整單公司優惠
基於（小計 - 整單業務優惠）計算
↓

第 7 層：總計（最終報價）
```

## ⚠️ 保護機制

### 單項優惠保護

1. **業務優惠不能超過原價**
```
原價：$70,000
業務優惠輸入：$80,000
→ ❌ 錯誤：優惠金額不能超過原價！
```

2. **公司優惠不能超過扣除業務優惠後的金額**
```
原價：$70,000
業務優惠：-$10,000
扣除業務優惠後：$60,000
公司優惠輸入：$70,000
→ ❌ 錯誤：優惠金額不能超過扣除業務優惠後的金額！
```

### 整單優惠保護

1. **整單業務優惠不能超過小計**
```
小計：$500,000
整單業務優惠輸入：$600,000
→ ❌ 錯誤：優惠金額不能超過小計！
```

2. **整單公司優惠不能超過扣除整單業務優惠後的金額**
```
小計：$500,000
整單業務優惠：-$50,000
扣除整單業務優惠後：$450,000
整單公司優惠輸入：$500,000
→ ❌ 錯誤：優惠金額不能超過扣除整單業務優惠後的金額！
```

## 💡 使用建議

### 優惠策略建議

**單項優惠 vs 整單優惠：**

**使用單項優惠：**
- 針對特定商品有不同優惠幅度
- 某些項目贈送，某些項目打折
- 需要明確顯示每個項目的優惠

**使用整單優惠：**
- 所有項目統一優惠比例
- 簡化計算流程
- 快速調整總報價

**混合使用：**
```
範例：大訂單策略
1. 門鎖、木門等小項目：單項業務優惠（9折）
2. 麻將桌等大項目：單項公司優惠（$2,000/台）
3. 最後加碼：整單業務優惠（5%）
```

### 優惠順序建議

**建議順序：**
```
1. 先設定單項優惠（針對特定項目）
2. 確認小計
3. 再設定整單優惠（針對總價）
4. 最終確認總計
```

### 修改優惠

**單項優惠修改：**
- 再次點擊「業務優惠」或「公司優惠」按鈕
- 輸入新的優惠值即可覆蓋

**整單優惠修改：**
- 點擊優惠行右側的「修改」按鈕
- 或再次點擊「整單業務優惠」/「整單公司優惠」按鈕

**清除優惠：**
```
單項優惠：點擊「↩️ 恢復」→ 清除所有優惠
整單優惠：輸入 0 → 清除該優惠
```
