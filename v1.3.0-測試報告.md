## 📋 功能 1：公司贈送

### 使用情境

**場景：** 報價時，某些項目想以「公司贈送」方式送給客戶

**操作流程：**
```
1. 新增報價 → 步驟 1（填寫客戶資料）
2. 步驟 2（選擇項目）→ 選擇任意項目
3. 已選項目列表 → 看到每個項目後方有三個按鈕：
   - [🎁 贈送]（綠色）
   - [移除]（紅色）
4. 點擊「🎁 贈送」
5. 結果：
   - 單價顯示：「🎁 公司贈送」（綠色標記）
   - 小計變為：$0
   - 按鈕變為：「↩️ 恢復」（橘色）
6. 需要恢復原價時，點「↩️ 恢復」
```

### UI 設計

**贈送前：**
```
智能門鎖
單價：$7,000 / 組

[10] 組  $70,000  [🎁 贈送]  [移除]
```

**贈送後：**
```
智能門鎖
🎁 公司贈送

[10] 組  $0  [↩️ 恢復]  [移除]
```

### 技術細節

**數據結構：**
```javascript
{
  id: 'ITEM-002',
  name: '智能門鎖',
  price: 7000,          // 原價
  quantity: 10,         // 數量
  customPrice: 0,       // 當前單價（贈送後為 0）
  total: 0,             // 小計（贈送後為 0）
  isFree: true          // 贈送標記
}
```

**重要保護機制：**
- 贈送品修改數量時，小計保持 $0
- 恢復後，單價和小計自動計算
- 總價會正確扣除贈送品金額

## 📋 功能 2：智能預設數量

### 使用情境

**場景：** 新增報價時，填寫包廂數 10 間、開放桌 2 桌

**問題（舊版）：**
- 選擇「智能門鎖」→ 預設數量 1
- 需要手動改成 10（包廂數）
- 每個項目都要手動調整，麻煩！

**解決（新版）：**
- 選擇「智能門鎖」→ 自動預設數量 10 ✅
- 選擇「麻將桌」→ 自動預設數量 12（10+2）✅

### 智能規則

#### 規則 1：包廂數量項目
**預設數量 = 包廂數**

適用項目：
- ✅ 智能門鎖
- ✅ ONE客製木門
- ✅ 木門（包含「木門」的項目）
- ✅ 藍牙喇叭 / 藍芽喇叭
- ✅ 門牌燈
- ✅ 智能控電

**範例：**
```
包廂數：10 間
開放桌：2 桌

選擇「智能門鎖」→ 預設數量：10
選擇「藍牙喇叭」→ 預設數量：10
選擇「門牌燈」→ 預設數量：10
```

#### 規則 2：包廂+開放桌項目
**預設數量 = 包廂數 + 開放桌數**

適用項目：
- ✅ 麻將桌（包含「麻將桌」的項目）
- ✅ 麻將椅（包含「麻將椅」的項目）
- ✅ 門弓器

**範例：**
```
包廂數：10 間
開放桌：2 桌

選擇「ONE頂規麻將桌」→ 預設數量：12（10+2）
選擇「麻將椅一組4張」→ 預設數量：12（10+2）
選擇「門弓器」→ 預設數量：12（10+2）
```

#### 規則 3：其他項目
**預設數量 = 1**

不符合規則 1 和 2 的項目：
- 冷氣、地板、壁紙等（以坪數計算）
- 加盟費用（單次費用）
- 輕隔間、輕鋼架等工程項目

### 保護機制

**最小值保護：**
```
包廂數：0 間
開放桌：0 桌

選擇「智能門鎖」→ 預設數量：1（最小值）
```

即使包廂數和開放桌數都是 0，預設數量也不會是 0，確保至少為 1。

## 🚀 使用建議

### 填寫順序

**重要！先填包廂數和開放桌數：**
```
1. 填寫店名、地址
2. ✨ 填寫包廂數、開放桌數（智能預設數量依賴這個）
3. 選擇負責人

### 調整數量

智能預設數量只是**起點**，仍可手動調整：
```

實際需要：15（多買 2 組備用）
→ 手動改為 15 ✅
```

### 贈送策略

建議贈送小金額項目，保持大項目收費：
```
✅ 適合贈送：藍牙喇叭（$2,000）、門牌燈（$800）
❌ 不建議贈送：麻將桌（$23,800）、加盟費（$283,200）
