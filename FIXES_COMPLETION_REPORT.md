## 🚀 新功能使用指南

### 📝 編輯報價

1. **從列表編輯：** 在 dashboard-simple.html 點擊「✏️ 編輯」按鈕
2. **從檢視編輯：** 在 view-quote.html 點擊「✏️ 編輯」按鈕或按 E 鍵
3. **修改內容：** 支援修改所有欄位（姓名、電話、狀態、項目）
4. **儲存：** 點擊「💾 儲存修改」或按 Ctrl+S / Cmd+S

### 💰 價目表管理

1. **訪問頁面：** `price-table.html`（需加入導航選單）
2. **新增項目：** 點擊「➕ 新增項目」填寫表單
3. **編輯項目：** 點擊「✏️ 編輯」修改內容
4. **刪除項目：** 點擊「🗑️ 刪除」（需確認）
5. **篩選搜尋：** 使用上方篩選器和搜尋框

### 📋 價目表快速選擇

1. **新增項目時：** 點擊「📋 從價目表選擇」按鈕
2. **輸入關鍵字：** 例如「桌遊」、「桌子」
3. **選擇項目：** 若有多項結果，輸入編號
4. **自動填入：** 名稱、分類、單價、單位自動填入
