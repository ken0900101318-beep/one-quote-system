## 💡 實作內容

### 1️⃣ 價格列表顯示權限

**修改位置**：`renderItem()` 函數

**實作邏輯**：
```javascript
// 獲取當前用戶角色
const currentUser = JSON.parse(localStorage.getItem('currentUser') || '{}');
const canSeeCost = currentUser.role === 'admin' || currentUser.role === 'viewer';

// 只有主管和會計能看到成本
${canSeeCost && item.cost ? `<span style="color: #e53e3e;">成本: $${item.cost.toLocaleString()}</span>` : ''}
${canSeeCost && item.friendPrice ? `<span style="color: #38a169;">友情價: $${item.friendPrice.toLocaleString()}</span>` : ''}
```

**效果**：
- ✅ 主管/會計：顯示「售價 / 成本 / 友情價」三項
- ✅ 員工：只顯示「售價」

**視覺設計**：
- 售價：紫色（主色）
- 成本：紅色 `#e53e3e`
- 友情價：綠色 `#38a169`

---

### 2️⃣ 編輯表單權限控制

**修改位置**：
- 表單 HTML：加入 `id="costFormRow"` 和 `id="friendPriceFormRow"`
- `openAddModal()` 函數：新增項目時
- `editItem()` 函數：編輯項目時

**實作邏輯**：
```javascript
// 根據權限顯示/隱藏成本欄位
const costRow = document.getElementById('costFormRow');
const friendPriceRow = document.getElementById('friendPriceFormRow');
if (costRow) costRow.style.display = canSeeCost ? 'block' : 'none';
if (friendPriceRow) friendPriceRow.style.display = canSeeCost ? 'block' : 'none';
```

**效果**：
- ✅ 主管/會計：表單顯示「成本」和「友情價」欄位
- ✅ 員工：表單隱藏這兩個欄位

---

### 3️⃣ 初始化權限檢查

**修改位置**：腳本開頭

**實作邏輯**：
```javascript
// 權限控制：檢查是否能看到成本
const currentUser = JSON.parse(localStorage.getItem('currentUser') || '{}');
const canSeeCost = currentUser.role === 'admin' || currentUser.role === 'viewer';

// 根據權限隱藏表單欄位
window.addEventListener('DOMContentLoaded', function() {
    if (!canSeeCost) {
        const costRow = document.getElementById('costFormRow');
        const friendPriceRow = document.getElementById('friendPriceFormRow');
        if (costRow) costRow.style.display = 'none';
        if (friendPriceRow) friendPriceRow.style.display = 'none';
    }
});
```

**效果**：
- ✅ 頁面載入時自動根據角色初始化表單
